     - Classroom use

Choosing Your Platform
----------------------

**Start with ByByte nano if:**

* You're new to robotics
* Learning at home
* Limited budget
* Want quick results
* No special tools available

**Choose ByByte Mega if:**

* Have some experience
* Teaching a class
* Want more capabilities
* Building complex projects
* Have access to tools

.. tip::
   **Not sure?** Start with ByByte nano. You can always upgrade to Mega later!
