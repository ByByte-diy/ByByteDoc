Overview
========

ByByte nano is a beginner-friendly robotics platform designed for:

* **Individual learners** starting their robotics journey
* **Home education** without special equipment
* **Quick prototyping** and experimentation
* **Budget-conscious** students and hobbyists

Key Features
~~~~~~~~~~~~

* ✅ Easy assembly - no professional skills needed
* ✅ Affordable - under $30 total cost
* ✅ Complete functionality with minimal components
* ✅ Arduino Nano hardware compatible
* ✅ Common components - easy to source
* ✅ 60-120 minutes assembly time (for beginners)

What's Included
~~~~~~~~~~~~~~~

The basic ByByte nano kit includes:

* Arduino Nano board (mega328p old)
* ESP32-CAM board, optional (for camera and WiFi)
* HC-05 (HC-06, HC-02) Bluetooth module (for Bluetooth communication)
* Motor driver module (DRV8833)
* 2x DC motors with 45mm wheels on 3mm shaft (400 - 750rpm, 6v, N20 motors)
* Ultrasonic distance sensor (HC-SR04)
* Line following sensors (5 x TCRT5000 module)
* Battery holder (1 x 6F22 "Krona" battery holder)
* 6F22 li-ion battery
* PCB board
* MINI-360 DC Back (for 5v and 3.3v power supply)
* 2x WS2812B LEDs 5mm (headlights)
* IR receiver VS1838 (for IR remote control)
* Resistance light sensor (LDR)
* Passive buzzer
* IR distance sensor (SFH4545, TEFT4300)
* Some electronic components (resistor, capacitor, diode, etc.)
* Motor mount with screws and nuts
* Ball wheel with screws
