Educational Content
===================

Contributing lessons and educational materials to БАБАЙ.

.. warning::
   **ALL lessons MUST be reviewed and approved by educational moderators!**

Overview
--------

БАБАЙ lessons are designed by:

* Robotics teachers
* IT professionals
* Educational experts

All lessons follow **STEM approach**: Practice First, Theory Second

Lesson Structure
----------------

Each lesson must include:

Required Sections
~~~~~~~~~~~~~~~~~

1. **Learning Objectives**
   
   * What students will learn
   * Skills they will develop
   * Knowledge they will gain

2. **Prerequisites**
   
   * Required knowledge
   * Previous lessons needed
   * Hardware requirements

3. **Materials Needed**
   
   * Hardware components
   * Software tools
   * Time required

4. **Step-by-Step Instructions**
   
   * Clear, numbered steps
   * Photos/diagrams for each step
   * Expected results

5. **Code Examples**
   
   * Well-commented code
   * Explanations of key concepts
   * Variations/extensions

6. **Theory Section**
   
   * Explain concepts used
   * Why things work this way
   * Real-world applications

7. **Challenges**
   
   * Practice exercises
   * Troubleshooting tasks
   * Extensions for advanced students

8. **Assessment**
   
   * How to verify learning
   * Expected outcomes
   * Common mistakes

Requirements
------------

Educational Quality
~~~~~~~~~~~~~~~~~~~

✅ **Clear Learning Objectives**
   * Specific and measurable
   * Age-appropriate
   * Aligned with STEM goals

✅ **Structured Approach**
   * Logical progression
   * Build on previous knowledge
   * Practice before theory

✅ **Engaging Content**
   * Interesting examples
   * Real-world applications
   * Hands-on activities

✅ **Accessibility**
   * Clear language
   * Visual aids
   * Support different learning styles

Age Groups
~~~~~~~~~~

Lessons should target:

* **Beginner (8-10 years)** - Basic concepts, simple projects
* **Intermediate (11-14 years)** - More complex, introduce programming
* **Advanced (15+ years)** - Advanced concepts, custom projects

Content Guidelines
------------------

Language
~~~~~~~~

* Use clear, simple language
* Define technical terms
* Avoid jargon
* Provide examples

Safety
~~~~~~

Always include:

* Safety warnings
* Proper handling instructions
* Adult supervision notes (if needed)
* Emergency procedures

Code Quality
~~~~~~~~~~~~

* Well-commented
* Follow Arduino style guide
* Use meaningful variable names
* Include error handling
* Provide full working examples

Images and Diagrams
~~~~~~~~~~~~~~~~~~~

* High-quality photos
* Clear circuit diagrams
* Step-by-step visuals
* Before/after comparisons

Submission Process
------------------

1. **Lesson Proposal**

   Create GitHub Issue:
   
   .. code-block:: text

      Title: Lesson Proposal - [Lesson Name]
      
      Target Age: [8-10 / 11-14 / 15+]
      Duration: [30min / 1hr / 2hrs]
      Difficulty: [Beginner / Intermediate / Advanced]
      
      Learning Objectives:
      - Objective 1
      - Objective 2
      
      Topics Covered:
      - Topic 1
      - Topic 2
      
      Materials Needed:
      - Component list
      
      Brief Description:
      [What students will build/learn]

2. **Review by Moderators**
   
   Moderators check:
   
   * Educational value
   * Age appropriateness
   * Feasibility
   * Alignment with curriculum

3. **Develop Full Lesson**
   
   After approval:
   
   * Write complete lesson
   * Create all materials
   * Test with target age group
   * Take photos/videos

4. **Educational Review**
   
   Submit for review:
   
   * Educational experts review
   * Test with students
   * Gather feedback
   * Revise as needed

5. **Final Approval**
   
   Once approved:
   
   * Add to curriculum
   * Publish in documentation
   * Announce to community
